A clean and well-structured Java project demonstrating **Object-Oriented Programming** concepts such as classes, constructors, encapsulation, getters/setters, and object creation.

### 📋 What This Practical Covers

- **Program 1**: Travel Time Calculator (distance/speed → time)
- **Program 2**: Employee Class with 10% raise demonstration
- **Program 3**: Date Class with formatted display

---

## ✨ Features

- Proper **encapsulation** (private fields + getters/setters)
- Constructor initialization
- Input validation (monthly salary must be positive)
- Clean console output with formatting
- Fully tested with dedicated test classes
- Ready to run in IntelliJ IDEA
